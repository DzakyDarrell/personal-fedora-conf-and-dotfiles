## 📖 Overview

This repository houses a collection of personal configuration files (dotfiles) tailored for a Fedora Linux environment. It provides a curated set of settings for various system utilities and applications, aiming to optimize the user experience, improve performance, and streamline workflows. Whether you're looking for a highly customized shell experience, a performance boost for gaming, or efficient system monitoring, these dotfiles offer a solid foundation.

## ✨ Configurations Included

This repository provides custom configurations for the following key areas:

-   🐠 **Fish Shell**: A powerful, user-friendly command-line shell with syntax highlighting, autosuggestions, and tab completion. The included dotfiles provide a personalized and efficient shell environment.
-   🚀 **Fastfetch**: A highly customizable command-line system information tool. The configuration showcases system details in an aesthetic and informative manner.
-   🎮 **Gamemode**: Optimizations designed to improve gaming performance by tweaking system settings when games are running.
-   📊 **KDE System Monitor**: Custom layouts and settings for KDE Plasma's System Monitor, providing tailored dashboards for tracking system resources.
-   🧠 **ZRAM**: Configuration for ZRAM, a Linux kernel module that creates compressed block devices in RAM, effectively increasing available memory and improving system responsiveness, especially on systems with limited RAM.
